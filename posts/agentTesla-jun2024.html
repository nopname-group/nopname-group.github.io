Overview

AgentTesla is a commodity infostealer written in .NET, primarily targeting credentials from browsers, email clients, and FTP applications. This variant introduced minor obfuscation changes compared to previous builds, using a custom .NET loader to complicate static analysis.

Static Analysis

Initial triage

Sample loaded in Binary Ninja and dnSpy. The binary is packed with a custom .NET loader that decrypts and loads the actual payload at runtime via Assembly.Load().

; Loader stub — pseudocode
byte[] payload = AES_Decrypt(resource_blob, key, iv);
Assembly asm = Assembly.Load(payload);
asm.EntryPoint.Invoke(null, null);

String decryption

Strings are encrypted with a single-byte XOR key derived from a hardcoded seed. After identifying the decryption routine all strings were extracted statically.

Dynamic Analysis

Executed in an isolated Windows 10 VM with Wireshark, Procmon, and x64dbg. Observed behaviors:

  • Credential harvesting from Chrome, Firefox, Outlook
  • Keylogging via SetWindowsHookEx
  • Exfiltration via SMTP over port 587
  • Persistence via HKCU\Software\Microsoft\Windows\CurrentVersion\Run

Conclusion

This variant represents a minor evolution in AgentTesla's obfuscation approach. The custom .NET loader complicates static analysis but is straightforward to unpack with a breakpoint at Assembly.Load(). The provided YARA rule covers the loader stub and key behavioral strings.
